The purpose of this package is to observe the behavior of random generators.The package contains 3 code files:
  - main.c: main code
  - von_neuman.c: code relative to Von Neuman's random numbers generator
  - congruential.c: code relative to Linear Congruential Number Generator

A Makefile is also provided to make compilation easier.

you can also find a results.txt file which is basically the output of the program. You can call the program by typing: './executable'
